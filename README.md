# OlympicMedalsWidget

The widget shows up to 10 countries that have won the most medals of a given kind in the Olympics games.
It is exposed on the window object as a function OlympicMedalWidget that takes two parameters, first, an element id (requred), and second (optional) is the initial sort order, and can be one of gold, silver, bronze or total.

## Demo

A demo preview app is located at https://olympic-medal-widget.herokuapp.com/

## Develop

To start the development run

```
npm run dev
```

This command will start a development server on port 3000

## Test

For unit tests

```
npm test
```

For e2e tests with cypress

```
npm run e2e
```

## Build

```
npm run build
```
